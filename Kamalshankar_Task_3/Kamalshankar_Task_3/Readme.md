## About project 
- This project is on CSS Selectors. i have use multiple selectors in it like 
- ID selector
- Class selector
- Direct child selector 

### Heading part 
- 3 headings (h1, h2, h4) all get "dark blue" text
- 2nd heading also has a green background through the ID Selector
- I use span tag to fill red color in "head" text

### Paragraph part
- There are 3 paragraphs and all three has white color in text and 20px font-size
- In 2nd paragraph i use ID selector to fill the green color in text. And in 3rd Paragraph the word "Paragraph" has higlightes with black color by using span tag.

### List Part
- I had to style the "here" Links without using classes or IDs. So i used: 
- direct child selector to target the first link and second link.

### run this file 
- first download both files (index.html + style.css), keep them together, open index.html in any browser. That's it!
